## 営業提案時の見せ方

最初にファーストビューを見せ、既存サイトよりも「何の会社か」「どこが強みか」が短時間で伝わる点を説明します。次に、住宅顧客だけでなく法人相談、協力会社、採用候補者にも届く導線整理を見せると提案意図が伝わりやすくなります。

施工写真ギャラリーは実績を捏造せず、実写真に差し替える前提で構成しています。写真が入ると完成度が大きく上がることを伝えてください。
