## 営業提案時の見せ方

1. まずファーストビューで、既存サイトより写真とコピーの印象が強くなる点を見せる。
2. 次に「適正価格」「無料見積」「住宅模型確認」を、会社の強みとして整理したことを説明する。
3. 施工例は実在しない案件名を作らず、実写真差し替え前提であることを伝える。
4. スマホ表示で、電話相談とフォーム相談の導線が自然に見える点を確認してもらう。
