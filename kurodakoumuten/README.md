# kurodakoumuten demo site

このフォルダは、黒田工務店様への営業提案用サンプルサイトです。公式サイトとして公開するものではありません。

## デザインコンセプト

リフォーム専門系をベースに、老舗工務店らしい落ち着きと職人感を加えた構成です。屋根修理・防水工事・外壁塗装の相談導線を中心に、写真、余白、細い罫線で上質に見せる方向にしています。

## 営業提案時の見せ方

既存サイトの情報は変えずに、写真の見せ方、ファーストビューの分かりやすさ、電話・相談導線の整理で印象を高めたデモとして見せてください。特に「屋根の不安をすぐ相談できる」導線と、実写真へ差し替えた場合の伸びしろを説明すると伝わりやすいです。

## GitHub Pagesへアップロードするときの注意点

`demo` リポジトリの直下に `kurodakoumuten` フォルダをそのままアップロードしてください。`index.html` からの参照はすべて相対パスです。`.github`、`.gitignore`、`.nojekyll`、`robots.txt` は不要です。
