# sinyama-koumuten demo site

このフォルダは、有限会社しんやま工務店様向けの営業提案用サンプルサイトです。公式サイトではありません。

## デザインコンセプト

「自然素材・健康住宅系」をベースに、鹿児島の地域工務店らしい落ち着きと木の質感を重視しました。既存サイトにある竹炭、高気密・高断熱、天然素材・無垢材、水まわりリフォームの伴走姿勢を、写真主役の上質な1ページ構成へ再編集しています。

## 営業提案時の見せ方

既存サイトの「情報はあるが古く見える」状態を、写真、余白、相談導線で整理したデモとして提示してください。特にファーストビュー、サービス整理、施工事例ギャラリー、電話相談とフォームの自然な導線を比較すると改善点が伝わりやすくなります。
