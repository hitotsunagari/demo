# inaba-koumuten demo site

このフォルダは、株式会社稲葉工務店様向けの営業提案用リニューアルデモサイトです。公式サイトとして公開するものではありません。

## デザインコンセプト

選定コンセプト: 1. 自然素材・健康住宅系

公式サイトから読み取れる「福岡の建築家とつくる家」「漆喰の家」「風が通る家」「屋上緑化・菜園住宅」「simple & smart house」を軸に、生成り、深緑、木の質感、明朝系の見出し、大きな余白で構成しています。量産LP感を避け、建築雑誌や設計事務所サイトに近い静かな見え方にしました。

## 営業提案時の見せ方

最初にファーストビューを見せ、「福岡の建築家とつくる家」という強みが3秒で伝わることを説明します。その後、既存サイトの情報を減らすのではなく、余白、写真、短いコピーで再編集することで、問い合わせ前の信頼感を高められる点を見せてください。

施工事例は、今後実写真に差し替えることで最も印象が変わるエリアです。特に新築、リノベーション、ローコストをカテゴリ別に整理すると、閲覧者が相談内容を選びやすくなります。

## GitHub Pagesへアップロードするときの注意点

- `inaba-koumuten` フォルダごと `demo` リポジトリ直下へアップロードしてください。
- `.github/`、`.gitignore`、`.nojekyll`、`robots.txt` は不要です。
- 画像パスは `assets/...` の相対パスです。フォルダ名や画像名を変更しないでください。
- このデモには `meta robots noindex,nofollow` を設定しています。
- フォームはデモ用で、実送信されません。
