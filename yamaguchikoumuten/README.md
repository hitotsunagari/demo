## 営業提案時の見せ方

ファーストビューでは「何をしている会社か」が3秒で伝わるよう、床工事・大工工事の専門性を短いコピーで表現しています。既存サイトの情報量は保ちつつ、スマホで読みやすい導線に整理した点を比較して見せると効果的です。
