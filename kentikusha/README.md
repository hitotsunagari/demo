# 有限会社建築社 リニューアル提案用デモ

このフォルダは、GitHub Pages の `demo` リポジトリ配下にそのままアップロードするための営業提案用サンプルです。公式サイトではありません。

## デザインコンセプト

選定コンセプトは「4. デザイン住宅系」です。建築誌のように写真を大きく扱い、余白、細い線、生成りの背景、木の質感で、オーダーメイド住宅の上質さが伝わる構成にしました。

## GitHub Pagesへアップロードするときの注意点

- `kentikusha` フォルダごと `demo` リポジトリ直下にアップロードしてください。
- `.github`、ワークフロー、`.gitignore`、`.nojekyll`、`robots.txt` は不要です。
- 画像パスは `index.html` から見て `assets/...jpg` の相対パスです。
- `meta robots` は `noindex,nofollow` のままにしています。
