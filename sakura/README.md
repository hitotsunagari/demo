## 営業提案時の見せ方

- まずファーストビューで「住まいの困りごとを相談できる会社」と直感的に伝わる点を見せる。
- 次にサービス一覧を見せ、既存サイトの豊富なメニューを整理すれば問い合わせが増えやすいことを説明する。
- 施工事例部分は、現状は仮画像であることを伝え、実写真へ差し替えると信頼感が上がる点を提案する。
- 保育園・幼稚園向け導線を独立して見せることで、一般家庭以外の相談獲得にもつながることを説明する。
