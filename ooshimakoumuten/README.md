## 営業提案時の見せ方

ファーストビューで「佐世保の工務店」「新築から住まいの困りごとまで」がすぐ伝わる点を見せてください。次に、既存サイトで分かれている新築、リフォーム、介護リフォーム、簡易工事の導線を1ページ内で整理した点を説明すると、リニューアル後の分かりやすさが伝わります。
