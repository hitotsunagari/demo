## GitHub Pagesへアップロードするときの注意点

- `suenaga` フォルダごと、既存の `demo` リポジトリ直下へアップロードしてください。
- `.github`、`.gitignore`、`.nojekyll`、`robots.txt`、デプロイ用ワークフローは不要です。
- `index.html`、`style.css`、`script.js`、`assets`、`README.md` の相対パスを変更しないでください。
- このデモでは `meta robots` を `noindex,nofollow` にしています。
