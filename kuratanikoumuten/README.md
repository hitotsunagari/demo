# 倉谷工務店 リニューアル提案デモ

このフォルダは、倉谷工務店様向けの営業提案用サンプルサイトです。公式サイトではありません。

## デザインコンセプト

コンセプトは「リフォーム専門系」を軸に、老舗工務店らしい落ち着きと現場感を加えています。
既存サイトで確認できるリフォーム、リノベーション、外構、小さな修理、新築相談の幅広さを、写真主役の1ページサイトとして整理しました。

## 公開前に確認すべき情報

- メールアドレスの正確な表記
- 営業時間
- 定休日
- 対応エリア
- 建設業許可など、掲載可能な資格・許可情報
- 見学会、イベント、SNS、採用情報の有無
- 施工事例の実写真、正式な案件名、掲載許可
- 問い合わせフォームの送信先

## 営業提案時の見せ方

最初にファーストビューで「何の会社か」「相談先として見やすくなったこと」を見せます。
次に、サービス一覧と施工イメージを見せ、既存サイトでは伝わりにくい「外構から室内まで相談できる工務店」という整理を説明します。
最後に、電話・フォーム・会社概要の導線が自然につながることを見せると、改善効果が伝わりやすくなります。
