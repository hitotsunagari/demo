# matsuzaki demo site

このフォルダは、株式会社松崎工務店様向けの営業提案用デモサイトです。公式サイトではありません。

## デザインコンセプト

選択コンセプト: 6. 建設・鉄骨・土木系

既存サイトで確認できる「大規模改修工事」を中心に、マンション・ビル・商業施設・店舗の改修相談につながる法人向けの上質な1ページサイトとして設計しました。黒・グレー・生成りを基調に、写真と余白で現場品質を伝える方向です。

## 営業提案時の見せ方

ファーストビューで「大規模改修工事の会社」と一目で伝え、次に事業内容、選ばれる理由、施工写真、問い合わせ導線を順に見せると効果的です。既存サイトの弱点である情報整理と写真訴求を、実写真に差し替えるだけで強化できる構成として説明してください。

## GitHub Pagesへアップロードするときの注意点

- `matsuzaki` フォルダごと `demo` リポジトリ直下へアップロードしてください。
- `.github`、`.gitignore`、`.nojekyll`、`robots.txt` は不要です。
- 画像パス、CSS、JS はすべて相対パスです。
- `meta robots` は `noindex,nofollow` に設定済みです。
- このページは営業提案用サンプルであり、公式サイトではない旨を画面端とフッターに明記済みです。
