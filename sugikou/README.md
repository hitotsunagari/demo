# sugikou demo site

このフォルダは、杉工務店様向けの営業提案用リニューアルデモです。公式サイトではありません。

## デザインコンセプト

選定コンセプト: 5. リフォーム専門系

既存サイトで確認できた「創業45年以上」「年間200件以上」「自社職人」「無料見積り」を、リフォーム相談のしやすさと現場対応力として見せる構成にしました。量産LP感を避けるため、写真を大きく使い、余白・細い罫線・木質感のある落ち着いた配色でまとめています。

## 既存サイトから反映した情報

- 会社名: 杉工務店
- 所在地: 福岡県筑紫野市周辺で確認。詳細住所は要確認
- 主なサービス: リフォーム、リノベーション、小さな修繕、無料見積り相談
- 強み: 創業45年以上、年間200件以上、自社職人、無料見積り
- 施工事例の傾向: 住宅リフォームを中心とした見せ方に整理
- 既存サイトの雰囲気: 親しみやすい地域工務店の印象
- 既存サイトの弱点: 写真の訴求力、スマホでの見やすさ、問い合わせ導線の整理、強みの見せ方に改善余地あり
- 見学会、イベント、SNS、採用情報: 要確認

## 営業提案時の見せ方

最初にファーストビューを見せ、既存サイトより写真とコピーで会社の強みが伝わることを説明します。次に「相談しやすさ」「自社職人」「年間相談件数」を短い導線で見せ、最後に実写真へ差し替えた場合の施工事例ページ展開を提案すると効果的です。

## GitHub Pagesへアップロードするときの注意点

- `sugikou` フォルダごと `demo` リポジトリ直下へアップロードしてください。
- `.github`、`.gitignore`、`.nojekyll`、`robots.txt` は不要です。
- 画像パスは `assets/filename.jpg` の相対パスです。
- `index.html` には `meta robots` の `noindex,nofollow` を入れています。
