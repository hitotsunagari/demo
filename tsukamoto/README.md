# 塚本工務店 リニューアル提案用デモ

このフォルダは、塚本工務店様への営業提案用サンプルサイトです。公式サイトとして公開するものではありません。

## デザインコンセプト

コンセプトは「リフォーム専門系」です。既存サイトで確認できる内装・外装・水回り・増改築・電気工事・外構対応を、写真主体で落ち着いた住宅会社らしい印象に再整理しました。親しみやすさは残しつつ、生成り、深緑、木の質感を使い、安価なLP感が出ないよう余白と線を中心に構成しています。

## 営業提案時の見せ方

ファーストビューで「姶良市のリフォーム会社」であることを短く伝え、次にサービス、資格、施工事例、相談導線へ流れる構成です。既存サイトの情報量を削らず、スマホで読みやすい順序に並べ替えた点を提案ポイントにしてください。写真はすべて仮画像のため、実施工写真に差し替えると説得力が大きく上がります。
