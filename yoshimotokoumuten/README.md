# yoshimotokoumuten demo site

このフォルダは、株式会社吉源工務店の既存ホームページリニューアル提案用デモサイトです。公式サイトではありません。

## デザインコンセプト

自然素材・健康住宅系と老舗工務店系を組み合わせた、木の質感、余白、写真を主役にした上質な1ページ構成です。既存サイトの情報量を整理し、「商品ではなく住空間」という思想が短時間で伝わるようにしました。

## 営業提案時の見せ方

最初にスマホ表示でファーストビューを見せ、写真の印象と問い合わせ導線の自然さを確認してもらうのがおすすめです。その後、施工事例の見せ方、会社概要、相談フォームの順に説明すると、既存サイトからの改善点が伝わりやすくなります。
